# Contribution Guidelines
Welcome to contribute to `apifree-ai-resource`! Here's how you can get involved:


## 1. How to Report Issues or Suggestions?
- Go to the **Issues** tab at the top of the repository
- Click **New issue**
- Select the appropriate template (if available), and provide clear details (e.g., include the link if reporting an invalid resource)


## 2. How to Submit New Content (e.g., new APIs/prompt templates)?
1. Click **Fork** in the top-right corner of the repository to copy it to your own account
2. In your forked repository, edit or add files (e.g., update `free-ai-apis.md` to add a new API)
3. After committing your changes, go to **Pull requests** in your forked repo â†’ click **New pull request**
4. Write a brief description (e.g., "Add XX free AI API"), then submit the PR and wait for review


## 3. Content Rules
- For new APIs: Must include a **working official link** + a short description (e.g., "Supports text generation")
- For prompt templates: Clearly state the **use case** + an example of the output
- For documentation: Use Markdown lists/headings to keep the formatting consistent
